# Flight Search Engine

Technical assessment project built with Next.js, TypeScript, and the Amadeus Self-Service API.

## Features
- Flight search by origin, destination, and date
- Real-time filtering by stops, airline, and price
- Price trend visualization
- Responsive design (mobile and desktop)

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Amadeus Self-Service API

## Notes
This project uses a server-side API route to securely interact with the Amadeus API and normalize flight data before it reaches the frontend.
